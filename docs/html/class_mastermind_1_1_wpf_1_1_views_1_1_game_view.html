Detailed Description

GameView.

Brugerflade til Mastermind-spillet, hvor spilleren kan gætte farver og starte nye runder.

Implementerer WPF UserControl og styrer spildata, evaluering af gæt og opdatering af brugerfladen. Arbejder sammen med OptionsRepository, SecretGenerator, Evaluering og IStatistikStore.
