Metoden sammenligner først hvert element i {\itshape gæt}  med det tilsvarende element i {\itshape secret}  for at finde sorte pegge (korrekt farve og position). Derefter gennemgås de resterende elementer for at finde hvide pegge (korrekt farve, forkert position), hvor der tages højde for allerede matchede positioner. Hvis længden af {\itshape gæt}  og {\itshape secret}  ikke er ens, kastes en Argument\+Exception.
