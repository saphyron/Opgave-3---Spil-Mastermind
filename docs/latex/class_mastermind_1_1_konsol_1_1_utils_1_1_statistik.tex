\doxysubsection{Detailed Description}
Håndterer statistik for Mastermind-\/spillet, herunder antal spil, sejre, tab, forsøg og win-\/streaks. 

Statistikken kan registrere resultater af spil og vise en oversigt baseret på de angivne Options. Denne klasse bruger Options til at tilpasse visningen baseret på spillets indstillinger, såsom sprog. Derudover beregnes statistik som gennemsnitlige forsøg ved sejr, bedste og værste forsøg, samt win-\/rate.
