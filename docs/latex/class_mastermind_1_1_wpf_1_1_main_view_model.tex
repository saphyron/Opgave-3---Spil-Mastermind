\doxysubsection{Detailed Description}
Hoved-\/viewmodel der skifter mellem visninger (spil, indstillinger, statistik) og håndterer exit. 

Implementerer INotify\+Property\+Changed og eksponerer kommandoer til navigation\+: {\ttfamily Nyt\+Spil\+Cmd}, {\ttfamily Options\+Cmd}, {\ttfamily Statistik\+Cmd}, {\ttfamily Exit\+Cmd}. Opretter og genbruger \doxylink{class_mastermind_1_1_core_1_1_persistence_1_1_options_repository}{Options\+Repository} og \doxylink{class_mastermind_1_1_core_1_1_persistence_1_1_json_statistik_store}{Json\+Statistik\+Store} og sætter {\ttfamily Current\+View} til den aktive WPF-\/view (User\+Control). Ved ændringer kaldes On\+Changed(string?) for at opdatere bindings.
