Feedback Mastermind.Core.Services.Evaluering.Evaluer ( Farve[] gæt,
Farve[] secret )

Evaluerer et gæt mod den hemmelige kode og returnerer antallet af sorte og hvide pegge.

Parameters
gætArray af farver, som repræsenterer spillerens gæt.
secretArray af farver, som repræsenterer den hemmelige kode.
Returns
Et Feedback-objekt med antal sorte og hvide pegge.

Metoden sammenligner først hvert element i gæt med det tilsvarende element i secret for at finde sorte pegge (korrekt farve og position). Derefter gennemgås de resterende elementer for at finde hvide pegge (korrekt farve, forkert position), hvor der tages højde for allerede matchede positioner. Hvis længden af gæt og secret ikke er ens, kastes en ArgumentException.
