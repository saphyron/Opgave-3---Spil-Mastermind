Detailed Description

Hoved-viewmodel der skifter mellem visninger (spil, indstillinger, statistik) og håndterer exit.

Implementerer INotifyPropertyChanged og eksponerer kommandoer til navigation: NytSpilCmd, OptionsCmd, StatistikCmd, ExitCmd. Opretter og genbruger OptionsRepository og JsonStatistikStore og sætter CurrentView til den aktive WPF-view (UserControl). Ved ændringer kaldes OnChanged(string?) for at opdatere bindings.

Mastermind.Wpf.MainViewModel.MainViewModel ( )

Initialiserer kommandoer, sætter startvisning og håndterer afslutning af app.

Opretter kommandoer til at skifte mellem Views.GameView, Views.OptionsView og Views.StatistikView. Starter med Views.GameView som aktiv visning. Exit-kommandoen lukker WPF-appen eller, hvis den ikke findes, kalder Environment.Exit(int).
