Detailed Description

OptionsView.

Brugerflade til at ændre spilindstillinger som længde, max forsøg, emojis og sprog.

Implementerer en WPF UserControl hvor indstillinger bindes direkte til felter og kan gemmes eller genindlæses via OptionsRepository. Indeholder kommandoer til Save og Reload uden brug af separat ViewModel.
