Detailed Description

StatistikView.

Viser en tabel med vedvarende spilstatistik og giver knapper til at nulstille og opdatere.

WPF UserControl der henter data fra IStatistikStore, aggregerer med StatistikTilføjer og formatterer output via kultur bestemt af OptionsRepository (da-DK / en-US). UI bindes til Table, og kommandoer (ResetCmd, RefreshCmd) løfter ændringer gennem DataContext-refresh.
