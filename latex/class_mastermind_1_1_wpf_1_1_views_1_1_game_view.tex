\doxysubsection{Detailed Description}
\doxylink{class_mastermind_1_1_wpf_1_1_views_1_1_game_view}{Game\+View}. 

Brugerflade til Mastermind-\/spillet, hvor spilleren kan gætte farver og starte nye runder.

Implementerer WPF User\+Control og styrer spildata, evaluering af gæt og opdatering af brugerfladen. Arbejder sammen med \doxylink{class_mastermind_1_1_core_1_1_persistence_1_1_options_repository}{Options\+Repository}, \doxylink{class_mastermind_1_1_core_1_1_services_1_1_secret_generator}{Secret\+Generator}, \doxylink{class_mastermind_1_1_core_1_1_services_1_1_evaluering}{Evaluering} og \doxylink{interface_mastermind_1_1_core_1_1_persistence_1_1_i_statistik_store}{IStatistik\+Store}.
