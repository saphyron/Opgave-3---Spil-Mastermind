\doxysubsection{Detailed Description}
\doxylink{class_mastermind_1_1_wpf_1_1_views_1_1_options_view}{Options\+View}. 

Brugerflade til at ændre spilindstillinger som længde, max forsøg, emojis og sprog.

Implementerer en WPF User\+Control hvor indstillinger bindes direkte til felter og kan gemmes eller genindlæses via \doxylink{class_mastermind_1_1_core_1_1_persistence_1_1_options_repository}{Options\+Repository}. Indeholder kommandoer til Save og Reload uden brug af separat View\+Model.
