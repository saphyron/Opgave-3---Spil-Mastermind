\doxysubsection{Detailed Description}
\doxylink{class_mastermind_1_1_wpf_1_1_views_1_1_statistik_view}{Statistik\+View}. 

Viser en tabel med vedvarende spilstatistik og giver knapper til at nulstille og opdatere.

WPF User\+Control der henter data fra \doxylink{interface_mastermind_1_1_core_1_1_persistence_1_1_i_statistik_store}{IStatistik\+Store}, aggregerer med \doxylink{class_mastermind_1_1_core_1_1_utils_1_1_statistik_tilf_xC3_xB8jer}{Statistik\+Tilføjer} og formatterer output via kultur bestemt af \doxylink{class_mastermind_1_1_core_1_1_persistence_1_1_options_repository}{Options\+Repository} (da-\/\+DK / en-\/\+US). UI bindes til {\ttfamily Table}, og kommandoer (Reset\+Cmd, Refresh\+Cmd) løfter ændringer gennem {\ttfamily Data\+Context}-\/refresh.
