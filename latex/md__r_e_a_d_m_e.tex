\hypertarget{md__r_e_a_d_m_e_autotoc_md6}{}\doxysection{\texorpdfstring{🧠 Hovedidé \& gameplay}{🧠 Hovedidé \& gameplay}}\label{md__r_e_a_d_m_e_autotoc_md6}

\begin{DoxyItemize}
\item Spillet genererer en hemmelig rækkefølge af farver (længde og forsøg er konfigurerbart via {\ttfamily Options}). ~\newline

\item Spilleren gætter kombinationer indenfor et maks. antal forsøg.
\item Feedback gives med {\bfseries{sorte}} (rigtig farve på rigtig plads) og {\bfseries{hvide}} (rigtig farve, forkert plads) markeringer.
\item Resultater gemmes og kan aflæses i {\bfseries{Statistik}} (WPF) og via konsol.
\end{DoxyItemize}

\begin{DoxyItemize}
\item {\bfseries{Domain}} ~\newline

 {\ttfamily Options\+Repository} gemmer/loader {\ttfamily Options} til/fra {\ttfamily Database/\+Options.\+json}. ~\newline
 {\ttfamily Json\+Statistik\+Store} implementerer {\ttfamily IStatistik\+Store} og gemmer resultater i {\ttfamily Database/\+Statistik.\+json}. ~\newline
 {\ttfamily Json\+File\+Paths} sørger for mappe/sti-\/opsætning og {\ttfamily Ensure\+Dir()}.
\item {\bfseries{Utils}} ~\newline
 {\ttfamily Statistik\+Tilføjer} – hjælpefunktioner ifm. statistik.

\item {\bfseries{MVVM}} ~\newline
 {\ttfamily Main\+View\+Model} (INotify\+Property\+Changed) holder {\ttfamily Current\+View} og eksponerer kommandoer\+: {\ttfamily Nyt\+Spil\+Cmd}, {\ttfamily Options\+Cmd}, {\ttfamily Statistik\+Cmd}, {\ttfamily Exit\+Cmd}.

 Let ICommand-\/implementering til bindinger.
\item {\bfseries{Lokalisering}} ~\newline
